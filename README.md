## Projekt & Teknisk- beskrivning

Vi har gjort en kalender med syfte att motivera en att komma igång med planering i ens liv. Det är en interaktiv kalender med todo-lista, dagens fokus, väder och bakgrundsmusik. Man kan lägga till och ta bort events och uppgifter, och allt sparas lokalt. Veckonummer och peppiga meddelanden visas snyggt med små animationer.

Det ska även ingå en tekniskt beskrivning, där ni beskriver en eller flera tekniska lösningar i projektet. Ni skall beskriva funktionen av den tekniska lösningen och hur den används i projektet. Ju tydligare beskrivning desto enklare att bedöma den tekniska lösningen. Beskrivningen skall vara förståelig även för personer som inte kan programmera.

### Externt producerade komponenter

I denna projekt så användes det externa material som en bild, mp3 fil för musiken och api för att för att få åtkomst till vädret.
